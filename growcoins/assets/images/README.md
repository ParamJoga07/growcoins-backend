# Assets Directory

## Logo File

Please place your `logo-growcoins.png` file in this directory.

The logo should be named exactly: `logo-growcoins.png`

The app will automatically use this logo in:

- Splash screen
- Login selection screen
- Home screen app bar
- Anywhere the `GrowcoinsLogo` widget is used
